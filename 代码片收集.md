- **JavaScript-随机数的生成**
  从一组有序的数据中生成一组随机并且不重复的数，类似于简单的抽奖程序的实现。
```JavaScript
var gRandomArr = function (arr, length) {
    // 使用sort将原数组的顺序打乱，让有序变成无序
    arr.sort(function () {
        return Math.random() - 0.5;
    });

    // 从原数组中一次性返回10个元素
    return arr.slice(0, length);
};
// 调用
gRandomArr(arr, 10);
```

  连续对两个数a和b进行三次异或运算，aˆ=b, bˆ=a, aˆ=b，可以互换它们的值

```JavaScript
var a = 10;
```

```csharp
// 半角转全角
public static string ToSBC(this string input)
        {
            char[] c = input.ToCharArray();
            for (int i = 0; i < c.Length; i++)
            {
                if (c[i] == 32)
                {
                    c[i] = (char)12288;
                    continue;
                }
                if (c[i] < 127)
                    c[i] = (char)(c[i] + 65248);
            }
            return new string(c);
        }

            {
                if (c[i] == 12288)
                {
```

- **先分组，然后每组里面仅取满足条件的行组 **
```SQL
var data = from n in test
    group n by n.ID into g
    let maxId = g.Max(a => a.order)
    from row in g
    where row.order == maxId
```
